# 1変数データの要約 - 分散と標準偏差

* データの代表値だけでは、データのばらつきを説明できない
* データのばらつきを散布度と呼ぶ
* 一般的によく使う散布度に分散と標準偏差がある

> ここでは標準偏差を算出する流れを解説します。

---

### （例1） 20人のテスト結果の度数分布表

``` 
84, 72, 86, 76, 68, 68, 72, 66, 78, 84, 74, 60, 86, 76, 74, 76, 70, 82, 76, 78
```

#### 平均

```
75.3
```
